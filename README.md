## Tools Installed

### 1. Cursor IDE
- Downloaded and installed Cursor from [cursor.com](https://cursor.com)
- Cursor is an AI-first code editor built on VS Code, designed to supercharge 
  development workflows using integrated AI assistance

### 2. Claude Code (Cursor Extension)
- Installed via Cursor's Extensions panel (Ctrl+Shift+X → search "Claude Code")
- Logged in using my Anthropic/Claude account
- Claude Code enables AI-assisted coding, documentation, and task execution 
  directly within the IDE

### 3. Codex (Cursor Extension)
- Installed via Cursor's Extensions panel (Ctrl+Shift+X → search "Codex")
- Logged in successfully
- Codex provides OpenAI-powered code generation and completion within Cursor

## Issues Encountered & How I Solved Them

No major blockers were encountered during this setup. The installation process 
for Cursor was straightforward, and both extensions were discoverable and 
installable within minutes via the Extensions panel.

The one thing I took a moment to verify was ensuring both extensions were 
properly authenticated — I confirmed active login status for both Claude Code 
and Codex before proceeding.
